# Road-Sense
ESP32 tabanlı otonom yol bozukluğu haritalama aracı. MPU6050 ivmeölçer verilerini Low Pass Filter ile işleyerek çukur/tümsekleri sınıflandırır. NEO-6M ile hassas konumlandırılan verileri SD karta loglar ve Leaflet.js destekli, internet bağımsız çalışan yerel Wi-Fi web arayüzü üzerinden interaktif ve canlı olarak görselleştirir.
